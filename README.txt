Jak uruchomić aplikacje?

W katalogu server  w terminalu uruchomić następujące komendy:

npm install

npm start

W katalogu client w terminalu uruchomić następujące komendy:
Na czas wykonania pierwszej komendy usunąć linijkę '"proxy": "http://localhost:5000"' z  pliku package.json

npm install

npm start
